## Currently Working on ...

### [Cuebase.online](https://github.com/Richard-luyt/CueBase)
- *Tech Stack: Next.js (Frontend), Express (Backend API), MongoDB.*
- A personal knowledge base system utilizing **RAG** with **dual searching mode**
  
  - **Strict Mode**: High-threshold vector matching. It’s for when I need the exact fact from my notes, period. No noise allowed.
    
  - **Explore Mode**: Lower similarity constraints to surface "latent" connections. It’s designed for brainstorming—finding that one obscure paper I saved six months ago that actually relates to what I'm writing now.
